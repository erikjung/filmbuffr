    FilmBuffr

    FilmBuffr provides a simple way to maintain a list of Netflix movie titles organized by director. Your list of movies will be saved automatically for your next visit. Enjoy!

    Made by Erik Jung for 10k Apart

    This application is using the Netflix OData API, which is currently still in production. If you notice any oddities with the movie results, this is likely the cause.
